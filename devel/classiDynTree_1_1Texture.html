◆ getPixels()

bool iDynTree::Texture::getPixels ( std::vector< PixelViz > &  pixels) const
virtual

Get the pixels of the texture.

Remember to call draw() first.

Parameters
pixelsThe output pixels. The size of the vector will be equal to the total number of pixels of the rendered texture, i.e. width of the texture times its height. Both these two parameters can be set in the textureOptions passed to the method ITexturesHandler::add. The pixels are saved in col-major format.
Note
This operation may affect the time performances of the visualizer, especially if the texture is large.
Returns
True in case of success, false otherwise
